# 🏦 FNB Data Curriculum – 16-Week Program

> Powered by **Capaciti** | Project Report 2025

This repository contains all weekly group projects and deliverables for the **16-Week FNB Data Analytics Programme**. The curriculum progresses from foundational data concepts through to a full Power BI capstone project.

---

## 📅 Curriculum Overview

| Week | Topic | Tools | Deliverable |
|------|-------|-------|-------------|
| [Week 1](./week-01/) | Intro to Data Analytics & SFIA Skills | Research | Presentation |
| [Week 2](./week-02/) | Data Collection & Management | Python, Power BI | Sales Summary |
| [Week 3](./week-03/) | Data Cleaning & Preparation | Power BI (Power Query) | `.pbix` + Screenshots |
| [Week 4](./week-04/) | Working with Databases | Power BI (DAX, Relationships) | `.pbix` + Screenshots |
| [Week 5](./week-05/) | Exploratory Data Analysis (EDA) | Power BI | `.pbix` + Screenshots |
| [Week 6](./week-06/) | Data Visualization | Power BI | Dashboard PDF |
| [Week 7](./week-07/) | DAX Basics | Power BI | KPI Dashboard |
| [Week 8](./week-08/) | Reporting & Automation | Power BI | Report Export |
| [Week 9](./week-09/) | Analytical Functions & What-if | Power BI (DAX) | A/B Report |
| [Week 10](./week-10/) | Segmentation & Grouping | Power BI (DAX) | Segment Report |
| [Week 11](./week-11/) | Data Engineering (Light ETL) | Power BI (Power Query) | ETL Report |
| [Week 12](./week-12/) | Data Governance & Lineage | Research + Power BI | Presentation |
| [Week 13](./week-13/) | Business Intelligence & Reporting | Power BI | Multi-page Report |
| [Week 14](./week-14/) | Advanced Power BI Features | Power BI | What-if Dashboard |
| [Week 15](./week-15/) | End-to-End Analytics Project | Power BI | Full Pipeline Report |
| [Week 16](./week-16/) | Capstone Project & Career Prep | Power BI | Final `.pbix` + Write-up |

## 🗂️ Repository Structure

```
fnb-data-curriculum/
├── README.md
├── week-01/          ← SFIA Research & Job Mapping
├── week-02/          ← Data Collection & Sales Summary
├── week-03/          ← Data Cleaning in Power BI
├── week-04/          ← Database Relationships in Power BI
├── week-05/          ← EDA – Student Scores
├── week-06/          ← Data Visualization Dashboard
├── week-07/          ← DAX Measures & KPI Dashboard
├── week-08/          ← Web Traffic Reporting
├── week-09/          ← A/B Testing Analysis
├── week-10/          ← Customer Segmentation
├── week-11/          ← ETL with Power Query
├── week-12/          ← Data Governance Research
├── week-13/          ← BI Executive Dashboard
├── week-14/          ← Advanced Features & What-if
├── week-15/          ← End-to-End Analytics Pipeline
└── week-16/          ← Capstone Project
```

Each week folder contains:
- `README.md` – Week objectives, tasks, and submission checklist
- `data/` – Dataset(s) used for the project
- `outputs/` – Final `.pbix`, PDFs, or exported files
- `screenshots/` – Dashboard/report screenshots

---

## 🛠️ Tools & Technologies

- **Power BI Desktop** – Primary BI and visualization tool
- **Python** – Data scripting (Weeks 2+)
- **Excel** – Supporting data work
- **DAX** – Data Analysis Expressions for Power BI measures
- **Power Query (M)** – ETL and data transformation
