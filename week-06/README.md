# Week 6 – Data Visualization in Power BI

## 🎯 Learning Objectives
- Apply data visualization best practices (charts, cards, KPIs)
- Design an effective single-page report layout
- Use bookmarks and drillthrough (optional)
- Export a Power BI report to PDF

## 📚 Visualization Best Practices

| Chart Type | Best Used For |
|-----------|---------------|
| Bar/Column Chart | Comparing categories |
| Line Chart | Trends over time |
| Card | Single KPI values |
| Donut/Pie | Part-to-whole (max 5 segments) |
| Slicer | Interactive filtering |
| Table | Detailed row-level data |

## 📋 Group Project – Monthly Sales Dashboard

### Dataset
- **File:** `data/Week-6-Sales-Monthly.csv` *(or Power BI Financial Sample)*

### Dashboard Requirements (1 Page)

- [ ] **Card:** Total Sales (formatted as currency)
- [ ] **Line chart:** Sales by Month
- [ ] **Bar chart:** Sales by Region
- [ ] **Slicer:** Product Category
- [ ] **Slicer:** Region
- [ ] Consistent color theme and report title
- [ ] Exported to PDF

### Design Tips

- Use a consistent color palette (match company/theme colors)
- Align visuals using gridlines
- Add a text box for report title and date
- Keep the layout clean — avoid cluttered visuals
