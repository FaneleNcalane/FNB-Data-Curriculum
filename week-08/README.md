# Week 8 – Reporting & Automation Features in Power BI

## 🎯 Learning Objectives
- Apply report themes, tooltips, and page navigation
- Export reports to PDF and PowerPoint
- Understand the Power BI Service publishing workflow (conceptually)
- Create a structured "Management View" page

## 📚 Key Features

| Feature | How to Access |
|---------|--------------|
| Report Theme | View → Themes |
| Tooltips | Format Pane → Tooltips → Report page tooltip |
| Page Navigation | Insert → Buttons → Navigation |
| Export to PDF | File → Export → Export to PDF |
| Export to PowerPoint | File → Export → Export to PowerPoint |
