# Week 1 – Introduction to Data Analytics & SFIA Skills

##  Learning Objectives

- Understand the four types of data analytics: Descriptive, Diagnostic, Predictive, and Prescriptive
- Learn the SFIA (Skills Framework for the Information Age) framework for data professionals
- Identify key tools used in data analytics: Java, Python, Excel, Power BI
- Map real-world job requirements to SFIA skill categories

##  Key Concepts

| Concept | Description |
|---------|-------------|
| Descriptive Analytics | What happened? (Historical data summaries) |
| Diagnostic Analytics | Why did it happen? (Root cause analysis) |
| Predictive Analytics | What will happen? (Forecasting/ML) |
| Prescriptive Analytics | What should we do? (Optimization/recommendations) |

### SFIA Skills Covered

| SFIA Code | Skill Name | Description |
|-----------|------------|-------------|
| DAAN | Data Analytics | Analysing and interpreting data to derive insights |
| DENG | Data Engineering | Building data pipelines and infrastructure |
| VISL | Visualisation | Creating visual representations of data |
| DATM | Data Management | Governing and managing data assets |
